# Code Linear Machine Test

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- GSAP (for animations)

## Folder Structure
- `app/`: Next.js application routes and global styles.
- `components/`: Reusable React components.
  - `ui/`: Base UI primitives (buttons, etc).
  - `layout/`: Layout wrappers (header, footer).
  - `features/`: Complex feature-specific components.
  - `animators/`: GSAP animation wrappers for components.
- `lib/`: Utility functions and GSAP configuration.
- `public/`: Static assets such as images and icons.

## Getting Started

### Installation
Install the project dependencies using npm:
```bash
npm install
```

### Running the App
Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
